🔒 Política de Privacidade - DF Studio Pro

Última atualização: 17 de Junho de 2026

1. Introdução

O DF Studio Pro é um aplicativo de processamento de áudio profissional desenvolvido por DF Studio. Esta política explica como coletamos, usamos e protegemos suas informações.

2. Dados Coletados

2.1 Permissão de Microfone

O aplicativo solicita acesso ao microfone do dispositivo exclusivamente para processamento de áudio em tempo real (Auto-Tune, efeitos, gravação). Nenhum áudio é enviado para servidores externos. Todo processamento ocorre localmente no dispositivo.

2.2 Dados de Áudio

Todas as gravações de áudio são salvas localmente no armazenamento do dispositivo. O aplicativo não faz upload de gravações para nenhum servidor.

2.3 Dados de Uso

Não coletamos dados de uso, análises ou telemetria.

2.4 Informações Pessoais

O aplicativo não coleta nome, email, localização ou qualquer informação pessoal identificável.

3. Uso dos Dados

Os dados de áudio são usados exclusivamente para:

  • Processamento de efeitos de áudio em tempo real
  • Correção de tom (Auto-Tune)
  • Gravação local de áudio
  • Reprodução de faixas importadas

4. Compartilhamento de Dados

Não compartilhamos, vendemos ou transferimos quaisquer dados para terceiros.

5. Armazenamento

As configurações do aplicativo (presets, marcadores, preferências) são salvas localmente no dispositivo usando SharedPreferences. Gravações são salvas no armazenamento externo do dispositivo.

6. Permissões do Dispositivo

O aplicativo solicita as seguintes permissões:

  • Microfone: Para captura e processamento de áudio
  • Armazenamento: Para salvar gravações e importar faixas
  • Bluetooth (opcional): Para dispositivos de áudio Bluetooth
  • USB (opcional): Para interfaces de áudio USB

7. Segurança

Implementamos medidas de segurança para proteger suas informações. Todo processamento é local e não há transmissão de dados pela internet.

8. Crianças

O aplicativo não é direcionado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos sobre alterações significativas.
